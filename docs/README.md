# Документация по проектной (учебной) практике

## Общая информация

Данная документация описывает процесс выполнения проектной (учебной) практики, включающей разработку и оформление учебного проекта «Говори. Учись. Играй», представляющего собой концепцию Telegram-бота для изучения русского языка иностранными студентами.

Практика выполнялась в рамках дисциплины «Проектная деятельность» с использованием инструментов Git, Markdown, HTML и CSS.

---

## Цель практики

Целью практики является получение практических навыков разработки и оформления программных проектов, включая:

- работу с системой контроля версий Git;
- создание и структурирование репозитория проекта;
- разработку статического веб-сайта;
- оформление проектной документации;
- представление результата в виде готового информационного продукта.

---

## Ход выполнения практики

В процессе выполнения практики были выполнены следующие этапы:

### 1. Подготовка среды разработки
- Создан репозиторий проекта на основе шаблона.
- Изучены базовые команды Git (clone, add, commit, push).

### 2. Структурирование проекта
- Организована структура репозитория в соответствии с требованиями.
- Созданы основные папки: `site/`, `docs/`, `reports/`.

### 3. Разработка веб-сайта
- Создан статический сайт на HTML и CSS.
- Реализованы страницы:
  - главная страница;
  - описание проекта;
  - информация об участниках;
  - журнал разработки;
  - ресурсы.
- Выполнено оформление дизайна и адаптация под различные устройства.

### 4. Документирование проекта
- Подготовлены материалы в формате Markdown.
- Описан процесс выполнения работы и структура проекта.

---

## Используемые технологии

- Git — система контроля версий
- HTML — структура веб-страниц
- CSS — оформление и стилизация
- Markdown — документация проекта

---

## Результаты

В результате выполнения практики был создан структурированный проект, включающий:

- веб-сайт, представляющий концепцию Telegram-бота;
- документацию в формате Markdown;
- организованный репозиторий с исходными файлами.

---

## Вывод

Практика позволила получить базовые навыки работы с инструментами разработки и закрепить знания в области веб-технологий и структурирования программных проектов.
