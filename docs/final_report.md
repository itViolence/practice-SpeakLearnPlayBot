# Финальный отчёт по проекту  
## Telegram-бот и образовательный сайт «Говори. Учись. Играй»

---

## 1. Общая информация о проекте

Проект представляет собой образовательную систему для изучения русского языка как иностранного. Он включает:

- многостраничный сайт с информацией о проекте;
- мини-игру для обучения через практику;
- систему шаблонизации HTML страниц на Python;
- структуру для генерации страниц из данных;
- документацию проекта.

---

## 2. Цель проекта

Цель проекта — разработка интерактивной образовательной платформы, которая помогает пользователям изучать русский язык через:

- теорию (страницы сайта);
- практику (мини-игра);
- интерактивное взаимодействие (логика и генерация контента).

---

## 3. Используемые технологии

В проекте использованы следующие технологии:

- HTML5 — структура сайта
- CSS3 — оформление интерфейса
- JavaScript — мини-игра
- Python — шаблонизатор и генерация страниц
- JSON — хранение данных страниц
- Git — контроль версий и разработка

## 5. Реализованная функциональность

### 5.1 Сайт проекта

- Главная

- Журнал
- Ресурсы
- Партнёры
- Игра
- Отчёт

---

### 5.2 Шаблонизатор

Реализован Python-движок, который:

- читает JSON данные;
- подставляет их в HTML шаблон;
- формирует итоговые страницы.

---

### 5.3 Мини-игра

Добавлена интерактивная игра:

- 5 вопросов;
- ввод ответа пользователем;
- подсчёт результата;
- система оценки:
  - 4–5 правильных ответов → зачёт
  - меньше → не зачёт
- возможность повторного прохождения.
