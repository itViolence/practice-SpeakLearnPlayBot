# 📌 Проект: Говори. Учись. Играй

## 🎯 Цель проекта

Разработка Telegram-бота и веб-платформы для изучения русского языка как иностранного с использованием игровых механик и интерактивного обучения.

---

## 🧠 Описание проекта

Проект представляет собой образовательную систему, включающую:

- Telegram-бот для изучения русского языка
- Веб-сайт с информацией о проекте
- Интерактивную мини-игру для проверки знаний

---

## ⚙️ Используемые технологии

- HTML5
- CSS3
- JavaScript
- Python (генерация страниц)
- Telegram Bot API (в перспективе / логика проекта)

---

---

## 🎮 Модификация проекта (игра)

Была добавлена интерактивная мини-игра:

- 5 вопросов по лексике
- пошаговое прохождение
- ввод ответа пользователем
- автоматическая проверка
- система оценки результата

### 📊 Система оценки:

- 4–5 правильных ответов → ЗАЧЁТ
- 0–3 → НЕ ЗАЧЁТ

Также реализована возможность повторного прохождения без перезагрузки страницы.
