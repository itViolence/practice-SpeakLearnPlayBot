# Отчёты в бинарных форматах

В данном разделе представлены итоговые отчёты по проектной (учебной) практике.

## Состав отчётных материалов

- **practice_report_template.docx** — основной отчёт по практике в формате Microsoft Word. Содержит полное описание выполненной работы в соответствии с требованиями шаблона.
- **practice_report_template.pdf** — итоговая версия отчёта в формате PDF, предназначенная для просмотра и передачи в электронном виде.

## Описание

Отчётная документация оформлена в соответствии с требованиями проектной (учебной) практики и включает:

- введение и постановку задач;
- описание деятельности заказчика;
- описание выполненных этапов работы;
- результаты практики;
- выводы и заключение.

Оба файла являются эквивалентными версиями итогового отчёта и содержат одинаковую информацию в разных форматах представления.
